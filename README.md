# InternTrail - Internship Finder for Freshers

## Problem Statement
Students struggle to find internships specifically for freshers. LinkedIn and Indeed mix senior roles with internships, making it hard for students to find entry-level opportunities.

## Solution
InternTrail is a simple web app that shows only remote and onsite internships for students and fresh graduates.

## Features
- View internship cards with company, title, type, stipend, duration
- Filter by: All Internships, Remote Only, Onsite Only
- Click "View Details" for full company description and requirements
- Apply Now button opens external application link
- Fully responsive (works on mobile, tablet, desktop)
- Modal closes with X button, Escape key, or clicking outside

## Tech Stack
- HTML5
- CSS3 (CSS custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript
- Kiro AI IDE for development

## How to Run Locally
1. Clone this repository
2. Open index.html in your browser
3. Start browsing internships!
